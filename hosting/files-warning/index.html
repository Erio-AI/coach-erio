What you'll need to do yourself

  1. Install Node.js on your computer Free, from nodejs.org — but you'll need to know how to install command-line software.
  2. Open Terminal / Command Prompt Then run npm install to fetch dependencies (a few hundred files), then npm run build to compile the site.

  5. Buy and connect your domain Purchase from GoDaddy, Namecheap, etc. Then edit DNS records (A, CNAME) — takes 24–48 hours to propagate.
